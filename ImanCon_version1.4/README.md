# ImanConnect

A JavaFX application that provides a platform for tracking Islamic faith-based activities, featuring user authentication, personal dashboard, and activity tracking.

## Features

- User authentication (Login/Sign Up)
- Personal dashboard with faith activity statistics
- Profile photo management
- Goal tracking system
- Activity progress tracking
- Settings management

## Prerequisites

- Java JDK 17 or later
- JavaFX SDK 21.0.2 or later

## How to Run

1. Clone this repository
2. Navigate to the project directory
3. Double-click `run.bat`

## Default Login

- Username: admin
- Password: admin

## Project Structure

- `src/main/java/com/faithapp/` - Java source files
  - `Main.java` - Application entry point
  - `controllers/` - FXML controllers
- `src/main/resources/` - Resource files
  - `fxml/` - FXML layout files
  - `images/` - Image resources
